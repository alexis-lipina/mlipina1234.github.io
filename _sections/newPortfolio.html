Have a look at some projects I've worked on.

EXALT

EXALT is a game I've been developing solo in the Unity engine since January 2018. Aside from the Rewired input package, I created the whole game, including art, programming and design. Some specific achievements I'm proud of are...

  • A real-time, fast-paced, strategic combat system which is easy to pick up but has significant depth
  • Custom AABB physics implementation built on top of Unity2D which supports
  • Custom shader which helps better communicate depth in an orthographic-camera platformer
  • Shadow script and shader which dynamically masks shadow sprites so they render directly on the geometry
  • Basic AI behaviour, including custom pathing with A*

Phantom Frame

Phantom Frame is a horror game in which the player must track and photograph invisible monsters using a magic polaroid camera. I designed and developed this game with my friend Davis Snider. Some contributions I've made to the project are...

  • Implemented ability to take photos using render targets and scene captures
  • Managing a 5-person multidisciplinary team of artists and developers as project lead
  • Integrated Wwise audio pipeline into project
