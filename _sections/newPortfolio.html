Have a look at some projects I've worked on.

EXALT

EXALT is a game I've been developing solo in Unity since January 2018. Aside from the Rewired input package, I created the whole game, including art, programming and design. Click here for more details! Some specific achievements I'm proud of are...

  • A real-time, fast-paced, strategic combat system
  • Custom 3D AABB physics built on top of Unity2D designed to support the unique visual style and 3D gameplay
  • Environment shader which applies a dynamic gradient to communicate depth in an orthographic-camera platformer
  • Shadow-casting shader which dynamically masks shadow sprites so they render directly on underlying geometry
  • Node-based AI navigation in 3D environment using adjacency lists and A*

Phantom Frame

Phantom Frame is a horror game in which the player must photograph invisible monsters using a cursed polaroid camera. I fill the roles of producer, art director, co-designer and programmer.

  • Managing a 5-person multidisciplinary team
  • Implementing gameplay code, such as the ability to take photos using render targets and scene captures
  • Integrated Wwise audio pipeline into project

JSniper

JSniper is a simple web game I made using the pixi.js graphics framework for a school project. Despite having a lot of old, bad jokes littered throughout, I'm proud of how the game feels to play. Play it here! (Sound warning!)
